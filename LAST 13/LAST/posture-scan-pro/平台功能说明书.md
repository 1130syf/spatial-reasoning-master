## 1. 系统概述

### 1.1 产品简介

**PostureScan Pro** 是一款专业的3D姿态评估桌面应用系统，基于先进的人工智能技术，通过普通摄像头即可实时捕获和分析人体姿态。系统广泛应用于医疗康复、运动训练、健身指导等领域，为专业人员提供科学、准确的姿态评估工具。

### 1.2 核心价值

| 价值点 | 说明 |
|--------|------|
| **非接触式检测** | 无需穿戴设备，使用普通摄像头即可完成检测 |
| **实时反馈** | 动作过程中实时显示关键指标和纠正建议 |
| **科学评估** | 基于人体工程学和运动科学的专业算法 |
| **数据追踪** | 完整记录每次评估，支持历史对比分析 |
| **易于使用** | 简洁直观的操作界面，无需专业培训 |

### 1.3 适用场景

- **医疗机构**: 康复评估、姿态矫正、术后恢复追踪
- **健身中心**: 动作指导、训练质量评估、会员体态分析
- **运动队**: 运动员体能测试、动作标准化训练
- **物理治疗**: 脊柱健康评估、姿势不良矫正
- **个人健康**: 家庭健康管理、自我体态监测

### 3.3 静态体态评估模块

#### 3.3.1 功能概述
静态体态评估模块通过摄像头捕获用户站立姿态，分析身体各部位的角度偏差和平衡状况，生成专业的体态评估报告。

#### 3.3.2 评估流程

```
开始评估
   ↓
启动摄像头 → 检测人体 → 显示骨骼标记
   ↓
校准姿态（可选）→ 记录中性位置
   ↓
实时分析 → 显示角度数据 → 显示平衡指数
   ↓
拍摄快照 → 保存评估数据
   ↓
生成报告 → 查看详细分析
```

#### 3.3.3 操作界面

```
┌─────────────────────────────────────────────────────────────┐
│  姿态捕获                    [设置] [校准] [拍照] [保存]    │
├──────────────────────┬──────────────────────────────────────┤
│                      │                                       │
│                      │  实时分析数据                         │
│                      │  ┌─────────────────────────────────┐ │
│    [摄像头画面]      │  │ 矢状面角度                       │ │
│                      │  │  头部前倾:  5.2°  [正常]        │ │
│    [骨骼叠加显示]    │  │  颈部角度:  2.1°  [正常]        │ │
│                      │  │  躯干角度: -3.5°  [警告]        │ │
│                      │  │  骨盆倾斜:  1.0°  [正常]        │ │
│                      │  └─────────────────────────────────┘ │

│                      │  ┌─────────────────────────────────┐ │
│                      │  │ 冠状面角度                       │ │
│                      │  │  肩膀水平:  2.3°  [警告]        │ │
│                      │  │  骨盆倾斜:  0.8°  [正常]        │ │
│                      │  └─────────────────────────────────┘ │

│                      │  │ 平衡指数                         │ │

│                      │  └─────────────────────────────────┘ │
└──────────────────────┴──────────────────────────────────────┘
```

#### 3.3.4 检测指标说明

**矢状面角度（侧面观察）**
| 指标 | 说明 | 正常范围 |
|------|------|----------|
| **头部前倾角** | 头部相对于肩部的前倾程度 | -5° ~ 5° |
| **颈部角度** | 颈椎的曲度 | -3° ~ 3° |
| **躯干角度** | 上身相对于垂直线的倾斜 | -5° ~ 5° |
| **骨盆倾斜角** | 骨盆前倾或后倾的程度 | -3° ~ 3° |

**冠状面角度（正面观察）**
| 指标 | 说明 | 正常范围 |
|------|------|----------|
| **头部倾斜** | 头部左右倾斜角度 | -2° ~ 2° |
| **肩膀水平** | 左右肩膀的高度差 | -3° ~ 3° |
| **骨盆倾斜** | 骨盆左右高低差 | -3° ~ 3° |

**平衡指数**
- **计算方法**: 综合各角度偏差计算，满分100分
- **评分标准**:
  - 90-100分: 优秀
  - 80-89分: 良好
  - 70-79分: 一般
  - 60-69分: 较差
  - <60分: 需要关注

#### 3.3.5 功能特性

**实时反馈**
- 摄像头画面实时显示
- 骨骼关键点实时标注
- 角度数据实时更新（每秒30-60次）
- 异常提示实时显示

**校准功能**
- **用途**: 消除个体差异，以当前姿态为中性位置
- **操作**: 保持标准站姿，点击"校准"按钮
- **效果**: 后续测量以此为基准计算偏差

**设置选项**
- 水平翻转: 镜像显示画面
- 显示骨骼: 开/关骨骼连线
- 显示关键点: 开/关关键点标记
- 显示角度: 在画面上标注角度数值

### 3.4 脊柱曲率检测模块

#### 3.4.1 功能概述
脊柱曲率检测专门用于评估脊柱侧弯情况，通过侧面观察测量脊柱的弯曲角度，及早发现脊柱健康问题。

#### 3.4.2 检测原理

```
侧面观察
   ↓
提取肩部中点坐标 (x1, y1)
   ↓
提取髋部中点坐标 (x2, y2)
   ↓
计算水平偏移量 = |x1 - x2|
   ↓
计算脊柱长度 = √[(x1-x2)² + (y1-y2)²]
   ↓
曲率角度 = arctan(偏移量 / 长度) × 180/π
   ↓
健康评估
```

#### 3.4.3 操作界面

```
┌─────────────────────────────────────────────────────────┐
│  脊柱弯曲度检测                                          │
├─────────────────────────────────────────────────────────┤
│                                                          │
│  说明: 请侧身站立，保持自然姿态                          │
│                                                          │
│  ┌───────────────────────┐                              │
│  │                       │                              │
│  │   [摄像头侧面画面]     │    [开始检测]               │
│  │                       │    [停止检测]               │
│  │   [脊柱线条标注]       │    [分析姿态]               │
│  │   - 黄色: 实际脊柱线   │                              │
│  │   - 白色虚线: 垂直参考线│                              │
│  │                       │                              │
│  └───────────────────────┘                              │
│                                                          │
└─────────────────────────────────────────────────────────┘
```

#### 3.4.4 评估标准

| 曲率角度 | 评估结果 | 建议 |
|----------|----------|------|
| **< 2°** | 脊柱曲率正常 | 保持良好姿态 |
| **2° - 5°** | 轻度侧弯，建议关注 | 加强核心训练，定期复查 |
| **> 5°** | 明显侧弯，建议就医检查 | 及时就医，进行专业评估 |

#### 3.4.5 检测报告

**报告内容**
- 曲率角度数值
- 水平偏移量
- 脊柱长度测量
- 健康评估结论
- 截图留存
- 改善建议

### 3.7 二头弯举训练功能

#### 3.7.1 功能概述
二头弯举训练模块监测手臂弯举动作，检测动作范围、大臂稳定性，提供专业的上肢训练指导。

#### 3.7.2 训练流程

```

   ↓
设定目标次数
   ↓
开始训练 → 站立准备
   ↓
手臂伸展 → 检测起始位置（角度 > 145°）
   ↓
弯举动作 → 监测肘部角度下降
   ↓
顶峰收缩 → 检测弯举高度（角度 < 65°）
   ↓
稳定性检测 → 监测肘部位移
   ↓
伸展还原 → 完成一次
   ↓
质量评分 → 显示反馈
   ↓
重复 → 直到达到目标
   ↓
生成报告
```

#### 3.7.3 检测指标

**动作范围检测**
| 指标 | 标准 | 说明 |
|------|------|------|
| **起始伸展度** | 肘部角度 > 145° | 手臂充分伸展 |
| **顶峰弯举度** | 肘部角度 < 65° | 充分弯举到位 |
| **完整ROM** | 角度变化 > 80° | 动作范围完整 |

**稳定性检测**
| 指标 | 检测方法 | 标准 |
|------|----------|------|
| **大臂稳定性** | 肘部位移量 | 位移 < 30% 臂长 |
| **身体摆动** | 躯干角度变化 | 躯干保持稳定 |

**评分标准**
- **优秀**: 伸展达标 + 弯举达标 + 大臂稳定
- **良好**: 动作范围达标，稳定性轻微问题
- **需改进**: 动作范围不足或大臂不稳定
- **无效**: 动作幅度太小或动作不规范

## 5. 数据报告说明

### 5.1 数据存储位置

**本地数据库**: 所有数据存储在本地MySQL数据库中
- 数据库名称: `posture_scan_pro_db`
- 存储内容: 人员信息、评估数据、训练记录
- 数据安全: 本地存储，不上传云端

### 5.2 数据隐私保护

- ✓ 所有数据本地存储
- ✓ 无需联网（模型首次加载需要网络）
- ✓ 不收集用户隐私信息
- ✓ 支持数据导出备份
- ✓ 删除人员时同步删除所有评估记录

### 5.3 报告数据解读

#### 5.3.1 静态体态评估数据

**角度数据**
- **正值**: 向右倾斜/前倾
- **负值**: 向左倾斜/后倾
- **绝对值**: 偏差程度

**平衡指数**
- **计算公式**: `100 - (偏差总和 × 权重系数)`
- **权重分配**:
  - 冠状面（左右平衡）: 60%
  - 矢状面（前后平衡）: 40%

**状态标识**
- 🟢 正常: 在正常范围内
- 🟡 警告: 轻微偏差，需注意
- 🔴 异常: 明显偏差，建议改善

#### 5.3.2 训练数据

**成功率计算**
```
成功率 = (有效次数 / 总尝试次数) × 100%
```

**质量评分标准**
- 深蹲: 深度 + 背部姿态
- 二头弯举: 动作范围 + 稳定性

## 6. 常见问题解答

**Q1: 摄像头无法启动？**
```
解决方案:
1. 检查摄像头是否被其他应用占用

3. 检查系统权限设置

   - macOS: 系统偏好设置 → 安全性与隐私 → 相机
4. 重启应用
```

**Q2: 摄像头画面卡顿？**
```
解决方案:
1. 关闭其他占用CPU的程序
2. 降低检测精度（设置 → 模型复杂度 → 选择0）
3. 确保摄像头驱动是最新版本
4. 检查电脑性能是否满足要求
```

**Q3: 检测不到人体骨骼？**
```
解决方案:
1. 确保全身在画面内
2. 增加环境光线
3. 穿着与背景颜色对比明显的衣服
4. 避免复杂的背景
5. 保持站立不动几秒等待检测
```

### 6.2 数据相关问题

**Q4: 数据库连接失败？**
```
解决方案:
1. 检查MySQL服务是否运行
   - Windows: 服务管理器中查看MySQL服务
   - macOS: 终端执行 mysql.server status
2. 验证.env配置文件
   - 数据库地址、用户名、密码是否正确
3. 确认数据库已创建
   - 数据库名: posture_scan_pro_db
4. 重启应用
```

**Q5: 删除人员后能否恢复？**
```
答案: 不能
说明: 
- 删除人员会同时删除该人员的所有评估记录
- 删除前会弹出确认提示
- 建议定期备份数据库
```

**Q6: 如何导出数据？**
```
方法:
1. 数据库导出
   mysqldump -u root -p posture_scan_pro_db > backup.sql

2. 报告截图
   - 在报告详情页面使用截图工具保存

3. 未来版本计划
   - 支持PDF导出
   - 支持Excel数据导出
```

**Q8: 为什么有些深蹲不计数？**
```
可能原因:
1. 深度不足（膝盖角度 > 100°）
2. 动作幅度太小（帧数 < 5帧）
3. 姿态严重错误
4. 检测丢失（离开画面）

解决方法:
- 查看每次的详细反馈
- 根据提示调整动作
- 确保全程在画面内
```

```
2. 等待报告生成
3. 然后再切换页面
```

### 6.4 评估相关问题

**Q10: 角度数据为什么一直变化？**
```
原因:
- 这是实时检测，会随姿态变化
- 人体自然会有微小晃动

建议:
1. 保持姿势稳定几秒
2. 使用"校准"功能消除个体差异
3. 拍照时保持不动
```

**Q11: 评估结果准确吗？**
```
准确性说明:
✓ 相对角度: 高精度（误差 < 2°）
✓ 趋势判断: 可靠
✓ 对比分析: 有效

注意事项:
- 摄像头角度会影响结果
- 建议多次评估取平均值
- 专业诊断请咨询医生
```

## 7. 使用技巧与建议

### 7.1 最佳使用环境

**光线**
- ✓ 充足的自然光或白光
- ✓ 避免逆光（背后有强光源）
- ✓ 避免侧光造成阴影

**背景**
- ✓ 简洁单色背景（浅色墙面最佳）
- ✓ 避免杂乱物品
- ✓ 避免镜子或反光物体

**服装**
- ✓ 贴身运动服
- ✓ 与背景颜色对比明显
- ✓ 避免宽松衣物遮挡关键点

**摄像头位置**
- 静态评估: 与胸部同高，距离2-3米
- 深蹲训练: 侧面，距离2-3米
- 二头弯举: 正面或侧面，距离2-3米

### 7.2 训练建议

**深蹲训练**
- 初学者: 目标5-10次，注重动作质量
- 中级: 目标10-20次，提高成功率
- 高级: 目标20+次，保持标准动作

**二头弯举训练**
- 使用适当重量（能完成目标次数）
- 优先保证动作规范
- 避免借力摇摆

**训练频率**
- 建议每周2-3次
- 每次间隔至少48小时
- 定期查看报告追踪进步

### 7.3 数据管理建议

**定期备份**
```bash
# 每周备份一次数据库
mysqldump -u root -p posture_scan_pro_db > backup_20251207.sql
```

**数据清理**
- 定期删除无用的测试数据
- 保留重要评估记录
- 导出重要报告截图
