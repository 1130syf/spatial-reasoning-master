## 1. 视频制作准备

### 1.1 拍摄环境准备

**硬件设备**
```
必需设备：
□ 电脑（满足系统要求）
□ 高清摄像头（1080p或更高）
□ 录屏软件（推荐OBS Studio或Camtasia）
□ 麦克风（用于配音）
□ 补光灯（可选，用于真人出镜）

可选设备：
□ 第二台显示器（用于监看）
□ 绿幕（如需抠图）
□ 手写板（用于标注）
```

**软件准备**
```
录制软件：
- OBS Studio（免费，推荐）
- Camtasia（付费，功能强大）
- ScreenFlow（macOS专用）

编辑软件：
- Adobe Premiere Pro（专业级）
- Final Cut Pro（macOS专用）
- DaVinci Resolve（免费版足够）
- 剪映专业版（简单易用）

辅助工具：
- PowerPoint/Keynote（制作片头片尾）
- Photoshop（处理截图）
- Audacity（音频处理）
```

**环境设置**
```
光线要求：
✓ 充足的自然光或柔和的白光
✓ 避免强烈的逆光和侧光
✓ 保持光线稳定，避免忽明忽暗

背景要求：
✓ 简洁的纯色墙面（白色或浅灰色）
✓ 无杂物和干扰元素
✓ 保持整洁专业的形象

音频环境：
✓ 安静无噪音的房间
✓ 关闭空调、风扇等噪音源
✓ 使用指向性麦克风
✓ 做消音处理（如铺地毯、窗帘）
```

### 1.2 应用程序准备

**数据准备**
```sql
-- 准备演示数据（在数据库中执行）

-- 1. 创建演示人员
INSERT INTO patients (name, gender, age, height, weight) VALUES
('张三', '男', 28, 175.00, 70.00),
('李四', '女', 32, 165.00, 55.00),
('王五', '男', 45, 180.00, 85.00);

-- 2. 确认运动项目数据存在
SELECT * FROM exercises;

-- 3. 清理旧的测试数据（可选）
-- DELETE FROM assessments WHERE created_at < DATE_SUB(NOW(), INTERVAL 7 DAY);
```

**界面设置**
```
应用设置：
□ 设置合适的窗口大小（建议1280x720或1920x1080）
□ 关闭开发者工具
□ 清理浏览器缓存
□ 测试所有功能确保正常运行
□ 准备好演示用的人员数据
□ 确保摄像头工作正常

系统设置：
□ 关闭系统通知（勿扰模式）
□ 隐藏桌面图标或使用干净背景
□ 关闭不相关的应用程序
□ 设置鼠标指针为高可见性样式
□ 调整屏幕分辨率为1920x1080
□ 关闭屏保和自动休眠
```

### 1.3 录制参数配置

**OBS Studio推荐设置**
```
视频设置：
- 基础分辨率: 1920x1080
- 输出分辨率: 1920x1080
- 帧率: 30fps或60fps
- 编码器: x264或NVIDIA NVENC H.264
- 码率: 6000-8000 Kbps

音频设置：
- 采样率: 48kHz
- 声道: 立体声
- 比特率: 192kbps

高级设置：
- 色彩空间: sRGB
- 色彩范围: 完整
- 关键帧间隔: 2秒
```

## 5. 后期制作指南

### 5.1 视频剪辑流程

**第一步：素材整理**
```
1. 创建项目文件夹
   PostureScan_Pro_Video/
   ├── 01_原始素材/
   │   ├── 截图/
   │   ├── 录屏/
   │   ├── 音频/
   │   └── 其他素材/
   ├── 02_标注素材/
   ├── 03_剪辑工程/
   └── 04_成品输出/

2. 素材命名规范
   - 使用有意义的文件名
   - 添加序号便于排序
   - 标注版本号（如有多个版本）

3. 素材分类归档
   - 按功能模块分类
   - 按用途分类（片头/正片/片尾）
```

**第二步：粗剪**
```
1. 导入所有素材到剪辑软件

2. 创建时间线序列
   - 序列设置：1920x1080, 30fps
   - 音频：48kHz, 16-bit

3. 按脚本大纲排列素材
   - 开场介绍
   - 界面总览
   - 核心功能演示
   - 特色亮点
   - 结尾总结

4. 删除冗余内容
   - 删除失误片段
   - 删除过长的等待时间
   - 删除重复内容

5. 调整顺序和节奏
   - 确保逻辑流畅
   - 控制单个片段时长
   - 添加转场预留
```

**第三步：精剪**
```
1. 精确剪辑
   - 在动作开始前0.5秒剪入
   - 在动作结束后0.5秒剪出
   - 使用J、L、K键精确定位

2. 添加转场
   - 场景切换：淡入淡出（0.5-1秒）
   - 功能切换：交叉溶解（0.3秒）
   - 特殊效果：推移、擦除（谨慎使用）

3. 调整速度
   - 加速：重复操作、等待过程（1.5-2倍）
   - 减速：关键动作、重点展示（0.5-0.7倍）
   - 暂停：重要画面（1-2秒）

4. 画面调色
   - 调整曝光：使画面明亮清晰
   - 调整对比度：增强视觉冲击
   - 色彩校正：统一色调
   - 锐化：增强清晰度（不要过度）
```

### 5.2 音频处理

**配音录制**
```
录音设备：
- 专业麦克风（推荐）
- 手机耳机麦克风（备选）

录音环境：
- 安静房间
- 关闭空调、风扇
- 铺设吸音材料

录音技巧：
- 保持嘴距麦克风15-20cm
- 语速适中，吐字清晰
- 语调自然，富有感染力
- 避免口水音、喷麦
- 每段录制多个版本备用
```

**音频编辑**
```
降噪处理：
1. 使用Audacity或Adobe Audition
2. 选择静音片段
3. 效果 → 降噪 → 获取噪音配置文件
4. 选择全部音频
5. 效果 → 降噪 → 应用
6. 降噪量：6-12dB

音量调整：
- 人声音量：-6dB至-3dB
- 背景音乐：-24dB至-18dB
- 音效：-12dB至-6dB

音频优化：
- 均衡器：增强中高频（人声清晰）
- 压缩器：平衡音量起伏
- 限制器：防止爆音

音频混合：
- 人声轨道
- 背景音乐轨道
- 音效轨道（点击、提示音）
- 导出：WAV或AAC 192kbps
```

**背景音乐选择**
```
音乐风格：
- 轻快、积极、科技感
- 避免过于激烈
- 避免歌词干扰

音乐来源：
- YouTube Audio Library（免费）
- Epidemic Sound（付费）
- Artlist（付费）
- AudioJungle（单曲购买）
- 自己创作

使用建议：
- 开场：节奏明快，吸引注意
- 正片：柔和背景，不抢主角
- 结尾：积极向上，留下印象
```
